#### 1. 继承的方式三种

- 公共继承    **public**
- 保护继承    **protected**
- 私有继承    **private**



#### 2. 继承中的对象模型

- 父类中所有的非静态成员属性都会被子类继承下去
- 父类中私有成员属性被编译器隐藏了，因此访问不到



#### 3. 继承中构造和析构的顺序

- 构造顺序 父类构造->子类构造->子类析构->父类析构



#### 4. 继承中同名成员处理方式

- 访问子类同名成员 **直接访问即可**
- 访问父类同名成员 **需要加作用域**
- 如果子类出现父类的同名的成员函数， 子类同名成员会隐藏父类同名成员函数，要加作用域

```c++
class Base{
public:
  Base(){ a = 10;}
  int a;
}
class Son:public Base{
  Son(){ a = 12;}
  int a;
}
Son s;
cout << s.a << endl;
cout << s.Base::a << endl;
```
